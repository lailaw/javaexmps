Flipflop - a simulation for the RS flip-flop written in Java
Version 1.00, September 2007
==================================================================

Runs on Mac OS/X, Windows any flavor, Linux.

Prerequisites:
-------------

Install the latest version of the Java Runtime Environment JRE 
(from http://java.sun.com)


Unpack the zip file. Copy flipflop.jar in any folder. Double-click on the
jar icon will start the program. For convenience, create a shortcut to run 
the program.

If this fails, create a shortcut that runs

java -jar flipflop.jar
